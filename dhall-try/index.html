Human-friendly

Dhall supports comments, multi-line string literals and string interpolation with non-technical users in mind

Tired of YAML?

Dhall eliminates the quirks that you know and "love"

You can also generate YAML from Dhall configuration files to slot right into your existing YAML-based tools

Get Started

Turing-completeness is not a feature

Dhall is a programmable configuration language, but Dhall is NOT Turing-complete

The Dhall configuration takes language security dead seriously in order to guarantee that Dhall programs never:

  • fail
  • hang
  • crash
  • throw exceptions
  • have any harmful side effects

You can safely import and evaluate untrusted Dhall code.

Learn More

Tough on messes

Dhall is built to handle large and weird configuration schemas

Don't let the type system fool you: Dhall can integrate with messy APIs and tools. You can even convert Dhall to ad-hoc and non-standard configuration file formats.
