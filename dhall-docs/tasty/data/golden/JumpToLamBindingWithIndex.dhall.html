On this example, we introduce the `a` variable in the two parameters of the
`fun` function and the jump-to-definition engine correctly highlights the usage
when indexes are used
-}

    λ(a : Text) ->
    λ(a : Text) ->
    [ a@1 ++ a@0 ]
